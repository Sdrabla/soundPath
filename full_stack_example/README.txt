this is a full stack example to use when creating it.

please install docker desktop to be able to use the dev commands to start it up check dev file for more info.

https://docs.docker.com/desktop/setup/install/windows-install

Notes:
The full stack wont operate unless docker has been opened on the ccomputer
./dev restart makes it restart the images on docker and run it
./dev down shuts down the docker container 
./dev clean removes volume data (Database files saved over)
